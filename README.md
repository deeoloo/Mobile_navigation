# Mobile Profile Card

A clean mobile interface with bottom navigation that toggles profile visibility.

![Preview](preview.png)

## Features
- Single-card design
- Profile toggle on icon click
- Responsive layout
- Minimal dependencies

## Quick Start
1. Clone repo
2. `npm install`
3. `npm run dev`

## Customize
- Edit `profile` object in `ProfileNav.jsx`
- Update styles in `ProfileNav.css`

## Tech
- React
- Material-UI Icons
- CSS
